E-Commerce Front Page (Mini Amazon)

A simple and responsive E-Commerce Web Application built using HTML, CSS, and JavaScript. This project simulates a mini online shopping platform where users can browse products, view details, add items to cart, and proceed to purchase.

* Features

* Product listing in responsive grid layout

* Product image preview with thumbnails

* Product detail popup modal

* Add to Cart functionality

* Buy Now option

* Remove items from cart

* Dynamic cart count update

* Cart data stored in localStorage (persists after refresh)

* Responsive design for different screen sizes

* Clean and modern UI

Technologies Used

 * HTML5

 * CSS3 (Flexbox & Grid)

 * JavaScript (Vanilla JS)

 * Browser LocalStorage
